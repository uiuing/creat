## 使用 👌


```shell
yarn install && yarn run dev
```

命令请查看 `package.json` 中的 `scripts`。

<br />


## 使用技术 📚

- 编程语言：TypeScript
- 渲染框架：React.js
- 状态管理：Recoil
- 本地缓存：IndexedDB | WebSQL | localStorage
- 样式渲染：Saas
- UI库：Semi Design UI

还有些可能没写上的的 .... 详情看：`package.json` 中的 `dependencies` 和 `devDependencies`。

- 代码规范：ESLint + Prettier + StyleLint
- 代码标准：Airbnb + Standard
